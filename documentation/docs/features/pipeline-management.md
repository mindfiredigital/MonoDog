# Pipeline Management

Monitor and control GitHub Actions and release pipelines directly from Monodog.

## Overview

- View recent pipeline runs and their statuses in a unified dashboard.
- Trigger, cancel or rerun GitHub workflows without leaving the app.
- Browse job logs and examine audit history for troubleshooting.
- Works across repositories using the GitHub Actions API.

## Key Features

### Dashboard

See the latest pipelines at a glance and filter by status or repository.

### Workflow Control

Start new runs, cancel in‑progress jobs, or rerun completed workflows when needed.

### Logs & Auditing

Fetch job logs and review who triggered an action and when it occurred.
